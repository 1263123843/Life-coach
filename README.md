# Life Coach AI 助手

这是一个基于DeepSeek R1 API开发的个人成长辅导网站，通过AI对话为用户提供个性化的建议和指导。

## 项目架构

### 前端结构
- `index.html`: 主页面，包含对话界面
- `styles.css`: 页面样式文件
- `main.js`: 前端交互逻辑

### 后端结构
- `server.js`: Node.js服务器，处理API请求和CORS

## 页面布局

### 主页面布局
- 顶部：标题栏
- 中间：对话内容显示区域
- 底部：消息输入框和发送按钮

### 样式特点
- 响应式设计，适配不同设备
- 简洁现代的界面风格
- 使用Flexbox布局实现页面结构

## 功能说明
1. 实时对话：用户可以与AI助手进行实时对话
2. 历史记录：保存对话历史
3. 流式输出：支持AI回复的流式显示

## 技术实现
- 前端：HTML5 + CSS + JavaScript
- 后端：Node.js + Express
- API：DeepSeek R1 API

## 安全措施
- API密钥安全存储
- 请求超时处理
- CORS安全配置

## 性能优化
- 响应式图片加载
- CSS压缩
- 代码分离和按需加载
